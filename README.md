# 3NC – Technische Veredelung & Smart Living

Website für 3NC – individuelle Smart-Home-Lösungen, galvanische Veredelung, 3D-Druck und Wohnraumoptimierung in Dresden.

## Seiten

- **index.html** – Startseite mit Hero und Service-Teaser
- **services.html** – Detaillierte Leistungen (Smart Living, 3D-Scan, Galvanik, Makeover, Spezial)
- **about.html** – Manufaktur: Philosophie, Skills, Arbeitsweise
- **contact.html** – Kontaktformular und Infos
- **impressum.html** – Rechtlich verbindliches Impressum (§5 TMG)
- **datenschutz.html** – DSGVO-konforme Datenschutzerklärung

## Tech Stack

- Statisches HTML/CSS/JS
- Fonts lokal gehostet (DSGVO-konform)
- Font Awesome via CDN (Cloudflare)
- Bilder generiert mit SDXL (lokales Modell)

## Deployment

Gehostet über **GitHub Pages** direkt aus dem `main` Branch.
